🤔 Що це взагалі таке?

Уявіть, що Claude Code — ваш цифровий помічник-програміст. «Голий» помічник вміє багато, але кожен проєкт особливий. Щоб не носити з собою «всі знання світу», помічник вчиться навичкам (скілам) і встановлює розширення (плагіни). Як додатки на телефоні: завантажили — отримали нову суперсилу.

🎯
Скіл (Skill)
Навичка — автоматично вмикається, коли Claude бачить релевантну задачу.
🔌
Плагін (Plugin)
Пакет з кількох скілів і команд. Зазвичай має тематику.
🛰️
MCP-сервер
Окрема програма, що дає Claude нові «руки» — підключення до зовнішніх сервісів.
⚡

Утиліта в терміналі (наприклад, gh), якою Claude користується через Bash.

Superpowers
⭐ 180 000
Дисципліна для Claude — змушує думати як інженер
🟢 Авто obra/superpowers

Найпопулярніший фреймворк скілів від Jesse Vincent. Перевизначає те, як Claude підходить до задач: спочатку обдумати, потім план, перевірити перед «готово». Десятки міні-скілів — брейнштормінг, тестування, систематичний дебаг, верифікація замість порожніх обіцянок. Якщо хочете щоб Claude робив роботу серйозно — ставте першим.

/plugin install superpowers@obra

🛰️ MCP-сервери

Окремі програми, які підключають Claude до зовнішніх сервісів.

🪡 Stitch (Google) Google product

Експериментальний MCP від Google для генерації UI-макетів. Скажіть «створи дизайн ландінгу» — отримаєте структурований макет.

claude mcp add stitch https://stitch.googleapis.com/mcp --transport http

🔗 n8n MCP workflow

Підключення до n8n — open-source платформи автоматизації. Створює, редагує, запускає ваші workflow.

npm install -g n8n-mcp
claude mcp add n8n npx n8n-mcp

📦 Офіційний реєстр MCP ⭐ 85 000

Колекція офіційних MCP: memory, filesystem, git, github, sqlite, postgres, puppeteer, brave-search.

npm install -g @modelcontextprotocol/server-memory
claude mcp add memory npx @modelcontextprotocol/server-memory

⌨️ CLI-інструменти

Claude може запускати будь-який CLI через Bash. Ось ті, що варто мати глобально.

🐙 gh — GitHub CLI офіційний

Робота з GitHub з терміналу: PR, issues, releases, workflows.

brew install gh
gh auth login

📊 gws — Google Workspace ⭐ 26 000

Drive, Sheets, Docs, Gmail, Calendar з терміналу.

npm install -g @googleworkspace/cli
gws auth login

▲ vercel ⭐ 15 000

Деплой проєктів на Vercel. Білд, env-змінні, домени за командою.

npm install -g vercel
vercel login

🔥 firecrawl-cli пара до плагіна

Командний інтерфейс до Firecrawl для скриптів і CI.

npm install -g firecrawl-cli

🎭 playwright-cli пара до плагіна

CLI до Playwright — автотести і скрейпінг скриптами без MCP.

npm install -g @playwright/cli

🛠️ Як встановити плагін?

Встановлення — три прості кроки. Скопіюйте команди в чат з Claude Code:

📥 Встановлення в три кроки
# 1. Додати маркетплейс (один раз)
/plugin marketplace add <автор>/<репозиторій>

# 2. Встановити плагін
/plugin install <ім'я-плагіна>

# 3. Перевірити встановлені:
/plugin list

💡 Більшість плагінів від Anthropic вже доступні без додавання маркетплейсу — просто /plugin install ім'я.

⚡ Рекомендований стек для початківця

Не намагайтесь поставити все одразу. Перевірений порядок:

  1. Базовий мінімум: Claude Code + Superpowers + Anthropic Skills. 80% задач.
  2. CLI-інструменти: gh, gws, vercel — глобально, авторизуватись.
  3. Для коду: code-review, frontend-design, claude-code-setup, skill-creator.
  4. Для веб-доступу: Firecrawl + Context7. Інтернет і свіжа документація.
  5. Браузерна автоматизація: Playwright (тести/CI) + Browser Harness (логіни/адмінки).
  6. Памʼять: Claude Memory.
  7. Інтерфейси: UI/UX Pro Max + Lazyweb + плагін design.
  8. Економія токенів: Caveman (увімкнути full).
  9. Мобільний контроль: Telegram Supercharged.
