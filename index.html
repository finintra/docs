🤔 Що це взагалі таке?

Уявіть, що Claude Code — це ваш цифровий помічник-програміст. «Голий» помічник вміє багато, але кожен проєкт особливий: одному потрібна допомога з дизайном, іншому — з документами, третьому — з аналізом даних.

Щоб не носити з собою «всі знання світу», помічник вчиться «навичкам» (скілам) і встановлює «розширення» (плагіни). Це як додатки на телефоні: завантажили — отримали нову суперсилу.

🎯
Скіл (Skill)
Спеціальна навичка — автоматично «вмикається», коли Claude бачить релевантну задачу. Як знати про мікроволновку, коли треба розігріти їжу.
🔌
Плагін (Plugin)
Пакет з кількох скілів і команд. Зазвичай має тематику (дизайн, документи, веб-доступ).
🏪
Маркетплейс
Сторінка-каталог, з якої можна встановлювати плагіни. Як App Store, але для Claude.
⚡
Слеш-команда
Команда, що починається з / — наприклад, /review. Швидкий спосіб запустити скіл вручну.

🛠️ Як встановити плагін?

Встановлення складається з трьох простих кроків. Просто скопіюйте команди в чат з Claude Code:

📥 Встановлення в три кроки
# 1. Додати маркетплейс (один раз)
/plugin marketplace add <автор>/<репозиторій>

# 2. Встановити плагін
/plugin install <ім'я-плагіна>

# 3. Готово! Перевірити встановлені:
/plugin list

💡 Більшість плагінів від Anthropic вже доступні без додавання маркетплейсу — просто пишіть /plugin install ім'я.

Playwright
⭐ 88 000 зірок (фреймворк) + 32k MCP
Браузерна автоматизація від Microsoft — найточніший контроль сторінок
🔵 Через MCP microsoft/playwright + microsoft/playwright-mcp

Промисловий стандарт для тестування і автоматизації веб-сторінок. Дозволяє Claude управляти Chromium, Firefox і WebKit через єдиний API: клікати, заповнювати форми, робити скріншоти, чекати на завантаження, перевіряти стан сторінок з мікросекундною точністю.

Через офіційний MCP-сервер Microsoft Playwright інтегрується прямо в Claude Code — і помічник може писати/запускати end-to-end тести, скрейпити сторінки з повним рендером, автоматизувати вхід у сервіси. На відміну від простих скрейперів, Playwright бачить сторінку точно як справжній користувач.

📥 Встановлення
# Через офіційний маркетплейс Anthropic (external_plugins):
/plugin install playwright

# Або як CLI глобально:
npm install -g @playwright/cli
Як використовувати: «протестуй цю форму у браузері», «зроби E2E тест для логіну», «перевір що сторінка відкривається у мобільному вигляді».

Context7
⭐ 55 000 зірок
Свіжа документація бібліотек прямо в Claude
🟢 Працює автоматично upstash/context7

Проблема: Claude вчили на даних, що могли застаріти на місяці. Документація React, Next.js, Tailwind змінюється швидко. Context7 розв'язує це — підтягує актуальну документацію бібліотек прямо в Claude під час роботи.

Замість галюцинацій про застарілі API ви отримуєте реальні приклади з останньої версії. Особливо корисно для популярних інструментів: React, Vue, Prisma, Tailwind, Django, Spring Boot.

# Через офіційний маркетплейс Anthropic:
/plugin install context7
Як використовувати: просто питайте про API будь-якої бібліотеки — Claude автоматично перевірить документацію.

Caveman Mode
⭐ 62 зірки
Економія токенів ~75% — Claude говорить як кам'яний геній
🔵 Через команди Shawnchee/caveman-skill

Викидає всі «зайві» слова з відповідей Claude — артиклі, ввічливості, hedging-фрази. Зберігає 100% технічного змісту, але економить ~75% токенів. Має рівні: лайт, повний, ультра.

Корисно для довгих сесій, коли контекст обмежений. Для кодів, комітів, попереджень про безпеку — автоматично перемикається в нормальний режим (auto-clarity), щоб уникнути двозначності.

📥 Встановлення
/plugin marketplace add Shawnchee/caveman-skill
/plugin install caveman
Корисні команди:
  • /caveman lite|full|ultraПеремкнути рівень стиснення
